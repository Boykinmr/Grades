PROJECT TITLE:			Grades

PURPOSE OF PROJECT:		This project elicits student and grade info from the user and then runs several tests on them to determine the capability of the student to pass the course. It includes how many points are needed, whether it is impossible or guaranteed to pass.

VERSION or DATE:		09/12/2017

HOW TO START THIS PROJECT:	No special instructions.

AUTHORS:			Michael R. Boykin

USER INSTRUCTIONS:		When prompted, enter the students name, points left to the end of the semester, points the student has earned so far and total points in the semester.  It then prints out how many points the student needs to get each letter grade, or if it is possible to get that grade.
